# Namoza Developer Assignment – OrthoNow

## Project Overview

This repository contains my submission for the Namoza Developer Assignment based on the OrthoNow healthcare project.

The assignment includes:

- Task 1 – Google Tag Manager (GTM) Event Tracking Schema
- Task 2 – Responsive Landing Page using HTML, CSS and JavaScript
- Task 3 – CRM & Google Ads Integration Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Tag Manager (Concept)
- Google Analytics 4
- HubSpot CRM
- Google Ads

---

## Task 1

Designed a complete GTM Event Schema including:

- Event Names
- Trigger Types
- Event Parameters
- GA4 Reports
- Booking Funnel Tracking
- dataLayer JSON Events
- Google Ads Conversion Recommendation

---

## Task 2

Developed a responsive healthcare landing page featuring:

- Hero Section
- Consultation Form
- JavaScript Validation
- Thank You Message
- Trust Badges
- Patient Testimonial
- Custom `window.dataLayer.push()` event

To run the project locally:

1. Download the repository.
2. Open `index.html` in any modern web browser.

---

## Task 3

Designed an end-to-end integration architecture connecting:

Landing Page
→ HubSpot CRM
→ Karix WhatsApp API
→ Google Tag Manager
→ Google Analytics 4
→ Google Ads

Also explained:

- Failure Point
- Fallback Strategy
- WhatsApp SLA Monitoring
